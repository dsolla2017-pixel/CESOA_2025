# Diogo Grawingholt - Portfólio Premium

Site de portfólio pessoal de Diogo Grawingholt, Gestor de TI e Especialista em Análise Data-Driven e IA.

## Acesso

- **Domínio personalizado:** [diogograwingholt.com](https://diogograwingholt.com)
- **GitHub Pages:** [dsolla2017-pixel.github.io/diogo-portfolio-site](https://dsolla2017-pixel.github.io/diogo-portfolio-site/)

## Tecnologias

- React + TypeScript
- TailwindCSS
- Vite (build)
- GitHub Pages (hospedagem)

## Deploy

O site é hospedado no GitHub Pages com deploy automático via GitHub Actions. Qualquer push na branch `main` aciona o deploy automaticamente.

## Estrutura

```
├── index.html          # Página principal (SPA)
├── 404.html            # Fallback para rotas SPA
├── CNAME               # Domínio personalizado
├── .nojekyll           # Desativa processamento Jekyll
├── assets/             # JavaScript, CSS e imagens
├── documents/          # Documentos (CV/PDF)
└── .github/workflows/  # GitHub Actions para deploy
```

## Manutenção

Para atualizar o site, faça as alterações nos arquivos e envie um push para a branch `main`. O deploy será feito automaticamente.
